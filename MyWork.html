    Build a Crowdfunding app with Django Rest Framework (DRF) and React

    This project is separated into two distinct projects:an API built using the Django Rest Framework - React was used to create the actual website for my Crowdfunding site. For Part A, I built the backend database and set up API end points using DRF, and tested them using Insomnia.

  • The GET method was set up to allow all projects to be viewed, as well as individual projects, to view all pledges and individual pledges, and to view user data.
  • The POST method was set up to allow a new user to register themselves on the site, to create a new project, to be able to pledge to a project, and to obtain a token for authorisation.
  • The PUT method was set up to allow a user to edit their own details, edit their project or allow pledges to be edited.
  • The DELETE method was set up to allow a user to delete their project, pledge and their own user account.
